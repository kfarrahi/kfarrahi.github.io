Undergraduate Projects Topics


      My research very generally considers methods from machine learning and network science to model various types of datasets. The majority of my work considers data mined by mobile phones, particularly location data and interaction data obtained by Bluetooth. Recent applications considered include studies in epidemiology and flu prediction modelling, as well as music recommenders.

      Possible project topics:

    1. Collect AIRBNB data (link) and run various forms of analysis.
    Examples include:

    (i) quantify and measure the supply vs demand to rank locations for investors

    (ii) predict bookings

    (iii) predict ratings


    2. Build a visualization tool for dynamical data

    3. Create a map visualiser to display cab GPS data obtained by CapSpotting or other suitable sites.

    4. Create a self-monitoring tool to summarise personal browsing activities (e.g. work vs social media)
